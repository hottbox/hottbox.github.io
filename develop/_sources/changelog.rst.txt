Changelog
=========

All notable changes to this project, that have been officially released,
can be found in the `CHANGELOG file <https://github.com/hottbox/hottbox/blob/master/CHANGELOG.md>`_
on **master** branch of our GitHub repository.

``hottbox`` is under active development, therefore, if you want to have a look
at the list of provisional changes for the next official release, visit
`CHANGELOG on develop <https://github.com/hottbox/hottbox/blob/develop/CHANGELOG.md>`_ branch.
