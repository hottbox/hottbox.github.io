kolda_fold¶

# Import statement

from hottbox.core import kolda_fold
kolda_fold(matrix, mode, shape)[source]¶

Fold a 2D array into a N-dimensional array.

Parameters

  • matrix (np.ndarray) – Unfolded version of a tensor

  • mode (int) – A mode along which original tensor was unfolded into a matrix

  • shape (tuple) – Shape of the original tensor before it was unfolded

Returns

tensor – N-dimensional array of the original shape

Return type

np.ndarray

Notes

  1. Much slower then fold

  2. At the moment it reverts unfolding operation (kolda_unfold). Will be generalised in a future
