make_clusters¶

# Import statement

from hottbox.datasets import make_clusters
make_clusters(dims, centers=5, n_samples=1000, center_bounds=(-10.0, 10.0), std=0.5, return_centers=False)[source]¶

Generates a tensor of any dimension with isotropic gaussian blobs as clusters

Parameters

  • dims (int) – specifies the order of the tensor

  • n_samples (int or list(int)) – Specifies the size of each clusters

  • centers (int or list(tuples)) – The number of clusters in the dataset and their size (can be a list)

  • center_bounds (tuple(float, float)) – Specifies the bound (min, max) for generating the centers

  • std (float) – The standard deviation of each of the generated clusters

  • return_centers (bool) – Returns the positions of the centroids

Returns

tensor – Generated tensor according to the parameters specified

Return type

Tensor
