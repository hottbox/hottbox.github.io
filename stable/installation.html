Installing HOTTBOXÂ¶

There are two options available:

  1. Install hottbox as it is from pypi.org by executing:

    pip install hottbox

  2. Alternatively, you can clone the source code which you can find on our GitHub repository and install hottbox in editable mode

    git clone https://github.com/hottbox/hottbox.git

cd hottbox

pip install -e .

    This will allow you to modify the source code in the way it will suite your needs

Note

If you wish to contribute, please, read the development guide and follow instructions therein
