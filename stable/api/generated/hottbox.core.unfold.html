from hottbox.core import unfold
unfold(tensor, mode)[source]¶

Unfolds N-dimensional array into a 2D array.

Parameters

  • tensor (np.ndarray) – N-dimensional array to be unfolded

  • mode (int) – Specifies a mode along which a tensor will be unfolded

Returns

matrix – Unfolded version of a tensor with a shape (tensor.shape[mode], -1)

Return type

np.ndarray
