Core Structures and Operations¶

This module contains methods for the most common operations within multilinear algebra and classes for the tensors represented through various tensor decompositions

hottbox.core.structures Classes for different tensor representations
Tensor(array) This class describes multidimensional data.
TensorCPD(fmat, core_values) Representation of a tensor in the CPD form.
TensorTKD(fmat, core) Representation of a tensor in the Tucker form.
TensorTT(cores, full_shape) Representation of a tensor in the TT form.
super_diag_tensor(order[, values]) Super-diagonal tensor of the specified order.
residual_tensor(tensor_A, tensor_B) Residual tensor
hottbox.core.operations Functions and operations for tensor algebra
khatri_rao(matrices[, skip_matrix, reverse]) Khatri-Rao product of a list of matrices.
kronecker(matrices[, skip_matrix, reverse]) Kronecker product of a list of matrices.
hadamard(matrices[, skip_matrix, reverse]) Hadamard product of a list of matrices.
unfold(tensor, mode) Unfolds N-dimensional array into a 2D array.
fold(matrix, mode, shape) Fold a 2D array into a N-dimensional array.
mode_n_product(tensor, matrix, mode) Mode-n product of a N-dimensional array with a matrix.

Kronecker product of a list of matrices.

  • reverse (bool) – If True, perform Kronecker product on the list of matrices in the reversed order
Returns:

result – The result of the Kronecker product is a matrix of shape (prod(n_rows), prod(n_columns) where prod(n_rows) = prod([m.shape[0] for m in matrices]) and prod(n_columns) = prod([m.shape[1] for m in matrices])

mode_n_product(tensor, matrix, mode)[source]¶

Mode-n product of a N-dimensional array with a matrix.

Parameters:

  • matrix (np.ndarray) – 2D array
  • mode (int) – Specifies mode along which a tensor is multiplied by a matrix. The size of a tensor along this mode should be equal to the number of columns of the matrix. That is: tensor.shape[mode] = matrix.shape[1]
