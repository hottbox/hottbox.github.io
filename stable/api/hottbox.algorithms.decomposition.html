Higher Order Orthogonal Iteration Decomposition.

Parameters:
  • init (str) – Type of factor matrix initialisation. Available options are hosvd.
  • process (tuple) – Specifies the order of modes to be processed. The factor matrices for the missing modes will be set to identity. If empty, then all modes are processed in the consecutive ascending order. Note, initialisation of a factor matrix that corresponds to the mode at the first position is skipped.
