Development guide¶

We appreciate any contributions to the hottbox. If you have suggestions for improvements, features or patches, please send us your pull requests (PR)! You can find specific instructions on how to issue a PR on github here. Feel free to open an issue if you find a bug or directly come chat with us on our gitter channel.
